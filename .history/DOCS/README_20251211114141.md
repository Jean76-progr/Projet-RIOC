# Documentation Globale
 
L’objectif de ce projet est de fournir un éditeur visuel permettant de créer des interfaces web modernes grâce à un système de glisser-déposer, avec génération automatique du code HTML/CSS.

## Architecture de la documentation

### 1. Editeur-Visuel
Ce projet contient l’application web développée en React.  
Il permet :
- de glisser-déposer des composants dans un canvas,
- de configurer visuellement des éléments (texte, style, position),
- de gérer des projets via IndexedDB,
- de générer automatiquement du code HTML/CSS,
- d’éditer le code via Monaco Editor.

### 2. Docs
Ce projet regroupe la documentation obligatoire :
- `README.md` : description des projets du groupe (ce fichier).
- `Explanation.md` : détails explicatifs supplémentaires (cadre juridique, contraintes non techniques, choix technologiques justifiés).

Chaque fichier de documentation est destiné à faciliter la compréhension du jury et à expliciter les choix architecturaux ou contextuels qui entourent le développement.
